函数 注
ones_like(a), zero_like(a), full_like(a,val) 根据数组a的形状创建数组
linspace(first,last,n) 根据起止数据等间距的填充数据，形成数组[first, last] [first, last)
concatenate() 将两个或多个数组合并成一个新的数组

函数 注
.reshape(shape) 不改变原数组
.resize(shape) 改变原数组
.swapaxes(ax1,ax2) 将数组n个纬度中的两个纬度进行调换
.flatten() 降维，返回折叠后的一维数组，原数组不变

ndarray数组的运算

  • 数组与标量之间的运算作用于数组的每一个元素
  • 一元函数
    对数组中每一个元素进行运算
    不改变x
函数 注
np.abs(x) np.fabs(x) 绝对值
np.sqrt(x) 平方根
np.square(x) 平方
np.log(x) np.log10(x) np.log2(x)
np.ceil(x) np.floor(x)
np.rint(x) 四舍五入
np.modf(x) 将数组个元素的小数和整数部分以两个独立数组形式返回
np.sin(x) np.cos(x) np.tan(x) np.sinh(x) np.cosh(x) np.tanh(x)
np.exp(x)
np.sign(x) 符号值
  • 二元函数
函数 注
+ - / * 两个数组各元素进行对应运算
np.maximun(x,y) np.fmax() np.minimun(x,y) np.fmin()
np.mod(x,y) 元素级的模运算
np.copysign(x,y) 将数组y中各元素值的符号赋值给数组x的对应元素
> < >= <= == != 算数比较，产生布尔型数组
