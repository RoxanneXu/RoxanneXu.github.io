VC维

Apr 22, 2017机器学习笔记点击

内容：2017年4月22日 星期六 关于VC维的整理。
说明：VC维（Vapnik-Chervonenkis Dimension）的概念是为了研究学习过程一致收敛的速度和推广性，由统计学理论定义的有关函数集学习性能的一个重要指标。

学习资料

  • VC Dimension

在机器学习里我们常常看到这样的说法：一般而言, VC维越大，学习能力就越强，学习也越复杂；可以通过 VC维计算学习风险的上界。那么VC 维到底是什么呢？

VC 维的定义

Let us say we have a dataset containing N points. These N points can be labeled in 2^N ways as positive and negative. Therefore, 2^N different learning problems can be defined by N data points. If for any of these problems, we can find a hypothesis h ∈ H that separates the positive examples from the negative, then we say H shatters N points. That is, any learning problem definable by N examples can be learned with no error by a hypothesis drawn from H. The maximum number of points that can be shatttered by H is called the Vapnik-Chervonekis (VC) dimension of H, is denoted as VC(H), and measures the capacity of the hypothesis class H.

数据集中有N个点，我们用将每个点标记为正类或负类，一共有$2^N$种标记方法。所以N个点可以定义$2^N$个不同的学习问题。对于其中任意一个问题，如果有假设h ∈ H可以将正类和负类区别开，我们说H可以打散(shatters)这N个点。也就是说，对N个点定义的任意学习问题来说，都可以通过H中的一个假设正确的学习。

H可以打散的点的最大个数，称为H的VC 维，记为VC(H)。

VC维在二维线性分类器上的计算示例

我们认为2D线性分类器的VC维为3，而不是4。

即，2D线性分类器可以打散集合大小为3的样本集合，不能打散有4个样本的集合。

无论4个点在一天直线上还是不在一条直线上（任意位置），都找不到一种情况能对2^4=16种标注进行打散。

比如四个点按下图方式排列，最后两种是不能被2D线性分类器所打散的。所以它不能满足VC维的要求。

总结，所以从上面可以看出，集合大小为3的样本集合是存在满足VC维条件的样本（只要存在就行，不要求所有的样本集合都要满足条件，例如3点一线就不满足）。而不存在大小为4的样本集合（注意：任意4个点就是一个大小为4的样本集合。同理于大小H个样本的集合）满足条件。所以说对于2D线性分类器的VC维为3。

VC 维的物理意义

VC 维可以反映假设H的强大程度(powerfulness)，VC 维越大，H也越强，因为它可以打散更多的点。

通过对常见几种假设的VC 维的分析，我们可以得到规律：VC 维与假设参数$\omega$的自由变量数目大约相等。
例如，对于2维感知机，$\omega=\omega_0,\omega_1,\omega_2$，有三个自由变量，dvc=3.

在N维空间中线性分类器和线性实函数的VC维是N+1

如果某函数的VC维无穷大，也就意味着，任意多个点无论怎样标注都能将其打散。例如sin(ax)。它可以将任意多样本的任意标注情况精确分开，即在训练集上达到100%的分类正确率。

VC维反映了函数集的学习能力，VC维越大则学习机器越复杂（容量越大），遗憾的是，目前尚没有通用的关于任意函数集VC维计算的理论，只对一些特殊的函数集知道其VC维。
